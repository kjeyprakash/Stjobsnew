Job Description

The Software Engineer is responsible for defining and developing software-based solutions as prescribed by the product management team in an Agile (Scrum) shop environment.  Using his or her inherent skills, the Software Engineer implements web-based software applications for use by external customers as well as Victory employees which provide solutions to defined problem domains.

Activities

Solution Definition - Proposes solutions to specific problem domains.

Scope Definition - Provides detailed description for the implementation of solution to proposed problem domain.

Estimation - Based upon the scope definition, provide accurate estimation in appropriate unit measurement of all tasks associated with the implementation of the solution as described in the scope definition.

Implementation - Proceed to perform all tasks as described in the scope definition, maintaining the highest quality standards, while constantly attempting to perform those tasks within the time limits as provided by the provided time estimations.

Progress Monitoring - Using issue and progress monitoring tools provided to team members, maintain an ongoing and timely description of current status of development efforts. Provide product and engineering management up to date description of any encountered obstacles or incidents that may compromise the completion of any assigned tasks, also in a timely manner.

 

Qualifications

  • Bachelors degree from accredited institute of higher learning in computer science, information science, or related field
  • Minimum 3 to 5 years experience working in a professional capacity as a software engineer, OR 5 to 7 years without a degree
  • Extensive experience using HTML and CSS
  • Extensive experience using one or more server-side programming languages such as Ruby, PHP, Java, or Python
  • Experience using an MVC-based web development framework such as Rails or Django
  • Professional software development experience with one or more of the following relational databases: MySQL, PostgreSQL, or Oracle
  • Very strong Javascript skills and professional experience with some of the more popular Javascript-based libraries like jQuery, Sencha Ext.Js, Ember.js etc. 
  • Experience working on teams of three or more developers using git, mercurial, or svn for source control
  • Experience working in the Agile (Scrum) project management environment
  • Strong debugging and general analytical skills and the ability to work in a fast-paced environment juggling multiple tasks
